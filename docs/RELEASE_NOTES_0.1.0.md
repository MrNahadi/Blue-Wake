# Release Notes: 0.1.0

## Highlights

- OpenCPN monitor window for own-ship EEXI/CII operational awareness.
- Toolbar toggle with persisted dashboard visibility.
- Vessel setup persistence through OpenCPN configuration.
- Year-to-date AER/cgDIST, CII rating, voyage totals, and CSV export.
- In-app Diagnostics view backed by severity-coded diagnostic events.
- Plugin-manager packaging script for the initial `msvc-wx32` / `x86` target.
- Synthetic replay validation fixture and packaging smoke test.

## Known Limitations

- Not for official regulatory submission.
- No class-society validation.
- No direct fuel-flow sensor integration.
- No IMO correction factors or full EEXI correction-factor model.
- Initial packaging path targets Windows `msvc-wx32` / `x86`; other platforms are future work.
